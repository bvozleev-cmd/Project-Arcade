Название: Project-Arcade (Economy Edition)
О проекте:
2D-аркада на Python с реализацией полноценной внутриигровой экономики и инвентаря. Проект демонстрирует интеграцию механик платформера с системами RNG (генератор случайных чисел) и менеджментом ресурсов.
Ключевые фичи:
1. Core Gameplay: Классический платформер со сбором коллекционных предметов (валюты).
2. Economy System: Сохранение прогресса валюты между уровнями, магазин предметов.
3. Lootbox Mechanic: Алгоритм взвешенной вероятности (Weighted Random) для открытия кейсов разной ценности.
4. Skin System: Динамическая смена спрайтов персонажа в зависимости от экипированного предмета из инвентаря.
