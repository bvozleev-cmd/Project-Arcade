# Project-Arcade (Economy Edition)

---

## О проекте

**Project-Arcade (Economy Edition)** — это 2D-аркадная игра, разработанная на языке **Python** с использованием библиотеки **Arcade**.  
Проект демонстрирует реализацию **полноценной внутриигровой экономики**, системы инвентаря и вероятностных механик (RNG), интегрированных в классический платформер.

Основной акцент сделан на сочетании:
- геймплейных механик платформера;
- менеджмента ресурсов;
- случайно-вероятностных систем наград.

---

## Ключевые фичи

### 1. Core Gameplay

- классический 2D-платформер;
- перемещение, прыжки, гравитация;
- сбор коллекционных предметов (внутриигровой валюты);
- прохождение уровней с сохранением прогресса.

---

### 2. Economy System

- единая внутриигровая валюта;
- сохранение валюты между уровнями;
- учёт максимального прогресса игрока;
- магазин с предметами и скинами;
- автоматическая разблокировка контента при достижении нужного количества ресурсов.

---

### 3. Lootbox Mechanic

- реализация системы кейсов (lootbox);
- использование **взвешенного генератора случайных чисел (Weighted Random)**;
- разные уровни редкости наград;
- контроль вероятностей выпадения предметов;
- возможность масштабирования системы наград.

---

### 4. Skin System

- система экипировки персонажа;
- динамическая смена спрайтов в реальном времени;
- связь с инвентарём игрока;
- сохранение выбранного скина между сессиями;
- визуальная индикация выбранного предмета.

---

## Назначение проекта

Проект создан с целью:
- демонстрации архитектуры 2D-игры на Python;
- практической реализации экономики в игре;
- работы с RNG и вероятностными алгоритмами;
- интеграции базы данных в игровой процесс;
- создания расширяемой и модульной игровой системы.

---

## Итог

**Project-Arcade (Economy Edition)** — это пример масштабируемого 2D-проекта, объединяющего платформер, экономику, инвентарь и случайные награды в единую систему, готовую к дальнейшему развитию и усложнению.
