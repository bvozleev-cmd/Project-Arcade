# Пояснительная записка  
## Проект: 2D платформер **Coin Quest**

---

## 1. Общая информация

**Coin Quest** — это 2D платформер, разработанный на языке **Python** с использованием библиотеки **Arcade**.  
Игра ориентирована на прохождение уровней, сбор кристаллов, открытие скинов персонажа и сохранение прогресса игрока.

В проекте реализованы:
- система уровней;
- сохранение прогресса;
- магазин скинов;
- несколько экранов интерфейса;
- физика платформера;
- звуковое сопровождение.

---

## 2. Используемые технологии

- **Язык программирования:** Python  
- **Игровая библиотека:** Arcade  
- **База данных:** SQLite  
- **Редактор карт:** Tiled (.tmx)  
- **Формат ресурсов:** PNG, MP3  

---

## 3. Структура проекта

Проект разделён на логические модули, каждый из которых отвечает за отдельную часть игры.

---

## 4. Описание файлов и их назначение

### `database.py`

Файл отвечает за работу с базой данных SQLite и хранение прогресса игрока.

#### Таблицы базы данных:
- `levels` — информация об уровнях
- `skins` — магазин скинов
- `player_skin` — выбранный скин игрока

#### Основной функционал:
- инициализация уровней и скинов;
- сохранение рекордов по кристаллам;
- разблокировка и выбор скинов;
- получение данных для UI и логики игры.

---

### `game.py`

Основной файл с игровой логикой.

#### Классы:
- **Player** — игровой персонаж, загрузка выбранного скина, разворот влево/вправо.
- **MyGame** — основной игровой экран.

#### Реализованные механики:
- движение и прыжки;
- гравитация;
- столкновения с платформами;
- сбор кристаллов;
- смерть при падении;
- победа при входе в дверь;
- режим GOD MODE;
- камера, следующая за игроком;
- визуальные эффекты частиц.

---

### `menu.py` (MenuView)

Главное меню игры.

#### Функции:
- запуск игры;
- переход в магазин;
- выход из игры.

---

### `level_select.py` (LevelSelectView)

Экран выбора уровней.

Отображает:
- список уровней;
- статус прохождения;
- количество собранных кристаллов;
- кнопку возврата в меню.

---

### `shop.py` (ShopView)

Магазин скинов персонажа.

#### Логика:
- подсчёт общего количества кристаллов;
- автоматическая разблокировка скинов;
- визуальное различие доступных и недоступных скинов;
- сохранение выбранного скина в базе данных.

---

### `pause_view.py` (PauseView)

Экран паузы.

Позволяет:
- продолжить игру;
- перейти к выбору уровня;
- вернуться в главное меню.

---

### `death_view.py` (DeathView)

Экран поражения.

Появляется при:
- падении за пределы уровня;
- столкновении со смертельными объектами.

---

### `sounds.py`

Централизованная загрузка всех звуков:
- кнопки;
- прыжок;
- сбор кристаллов;
- победа;
- поражение;
- смена скина.

---

## 5. Уровни и карты

- Карты создаются в **Tiled**
- Формат файлов: `.tmx`
- Каждый уровень — отдельная карта

### Используемые слои:
- `Wall` — коллизии
- `Platforms` — платформы
- `Items` — кристаллы
- `Door` — выход с уровня
- `Water/Lava` — опасные зоны
- `Player` — точка спавна

---

## 6. Взаимодействие с пользователем

Игрок:
- управляет персонажем с клавиатуры;
- выбирает уровни;
- собирает кристаллы;
- открывает и выбирает скины;
- взаимодействует с UI-кнопками;
- получает визуальную и звуковую обратную связь.

---

## 7. Особенности проекта

- сохранение прогресса между запусками;
- модульная архитектура;
- расширяемость;
- визуальные эффекты частиц;
- система рекордов;
- полноценный магазин скинов;
- разделение логики и интерфейса.

---

## 8. Заключение

Проект **Coin Quest** представляет собой полноценную 2D-игру с системой прогресса, магазином, несколькими игровыми экранами, физикой и сохранением данных.  
Архитектура проекта позволяет легко расширять игру, добавляя новые уровни, скины и механики.
